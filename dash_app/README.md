# SpaceX Launch Records Dashboard
This directory contains the interactive dashboard built using Plotly Dash. The application allows for real-time visual exploration of SpaceX launch records, enabling users to filter data by launch site and payload mass to discover trends in launch success.

## Features
1. Launch Site Filtering: A dropdown menu to switch between all sites or specific locations (CCAFS, KSC, VAFB).
2. Success Rate Analysis: Dynamic pie charts that show the ratio of successful to failed launches for specific sites, or the distribution of success across all sites.
3. Payload Correlation: An interactive scatter plot that displays the relationship between payload mass (kg) and landing success, color-coded by booster version.
4. Payload Slider: A range slider to filter data points based on specific weight classes.
